# Quote Reply (引用回复)

```
📸 1. screenshot → find target message
   2. right-click on the message bubble
📸 3. screenshot → find "Quote" or "引用" (NOT "Forward"/"转发") in context menu
   4. click "Quote" (or "引用" if Chinese UI)
   5. computer_batch: [type reply, key "Return"]
```

**5 calls, 2 screenshots.** Cursor auto-focuses in input after clicking Quote.

## Tips

- The context menu language depends on WeChat's UI language setting. Look for **"Quote"** (English) or **"引用"** (Chinese).
- Do NOT confuse with "Forward..." / "转发" — that opens the forward picker dialog instead.
