# Send a File or Image

**Cannot fully automate** — native file picker is inaccessible via Computer Use.

## Workaround

1. Ask user to copy the file in Finder (macOS) or Explorer (Windows).
2. Claude clicks the input box in WeChat.
3. Claude pastes with `cmd+v` (macOS) or `ctrl+v` (Windows).
4. Claude sends with `Enter`.

**2 calls, 1 screenshot** (after user confirms copy).
