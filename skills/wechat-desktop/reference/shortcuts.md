# Keyboard Shortcuts

| Action | macOS | Windows |
|--------|-------|---------|
| Send message | Enter | Enter |
| New line | Shift+Enter | Shift+Enter |
| Select all | Cmd+A | Ctrl+A |
| Paste | Cmd+V | Ctrl+V |
| Search | Cmd+F | Ctrl+F |

Enter/Shift+Enter can be swapped in WeChat settings (设置 → 通用 → 快捷键).
